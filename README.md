# Notes App

Aplikasi pencatatan sederhana berbasis web yang dibuat sebagai submission kelas Belajar Fundamental Front-End Web Development di Dicoding.

Project ini dibuat menggunakan JavaScript modern dengan pendekatan Web Component untuk membangun komponen UI yang reusable. Aplikasi mampu menampilkan daftar catatan, menambahkan catatan baru, serta menerapkan layout modern menggunakan CSS Grid.

## Demo Project

* Live Demo:
  https://rurinhaliza.github.io/Dicoding-Notes-App/

## Features

* Menampilkan daftar catatan
* Menambahkan catatan baru
* Menggunakan Web Component
* Layout menggunakan CSS Grid
* Responsive layout
* Tampilan modern dan sederhana
* Struktur project menggunakan module bundler

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Web Components
* CSS Grid
* Webpack


```bash id="0v72j4"

```

3. Jalankan project

```bash id="sm4k8k"

```
